# Repository

Bootstrapped with RepoPact, consumed from PyPI (`repopact==1.9.0`; see
`requirements-repopact.txt`). Run `pip install -r requirements-repopact.txt` once,
then validate with `python -m repopact_cli validate` (or `repopact validate`).
`schemas/` stay in-repo — RepoPact validates this repository's own contracts.
